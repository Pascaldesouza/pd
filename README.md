# Pascal DeSouza - Premium Portfolio

A clean, white, premium portfolio website built with HTML, CSS, and JavaScript.

## Features

- Premium white editorial design with sharp black typography
- Responsive layout across desktop, tablet, and mobile
- Hero intro animation, scroll reveals, subtle parallax, and refined hover effects
- Multi-page structure for Home, About, Projects, and Contact
- Inter Tight typography with strong hierarchy
- Image-led project showcase with filtering and modal details
- Contact form UI with validation behavior
- Professional downloadable CV PDF included

## Design Highlights

- White background, black typography, and soft gray supporting text
- Large confident hero headline
- Clean pill buttons and restrained navigation
- Premium project preview images
- Smooth motion that respects reduced-motion preferences
- Vanilla JavaScript, Font Awesome icons, and Google Fonts

## GitHub Pages

This is a static website, so it can be hosted directly from GitHub Pages.

1. Create a GitHub repository.
2. Upload all files and folders in this directory, including `assets/`, `portfolio-premium.css`, `portfolio-motion.js`, and the PDF files.
3. In GitHub, open `Settings` > `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose your main branch and the root folder `/`.
6. Save, then open the GitHub Pages URL once deployment finishes.

The site entry page is `index.html`.
